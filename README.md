# cineio-console-ios

cine.io end-user console app for the iOS App Store.

## Features

- display TOS and privacy policy
- sign-in with username and password
- sign-in via GitHub
- sign-out
- view list of projects
- view list of streams
- broadcast
- play
- update account
- add new project
- delete project
- add new stream
- delete stream
- universal app (iPhone / iPad)
